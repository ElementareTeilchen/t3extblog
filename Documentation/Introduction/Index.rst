What does it do?
================

This is a TYPO3 Extbase / Fluid extension which aims to replace t3blog. It is easy to implement and use.

T3ExtBlog has all lot of features needed for blogging:

- categories (usage of TYPO3-categories)
- tagging
- all TYPO3-content-elements could be used to create blog content
- commenting system
- antispam
- All views are organized in modules
- Views: list, single, latest, categories, archive, latest comments...


Visit & contribute: https://github.com/fnagel/t3extblog
